# Sharimar: Uma Jornada de Resgate e Inglês 📚⚔️

Este é um jogo educativo desenvolvido no **Godot 4.2** com o objetivo de ensinar conceitos básicos de língua inglesa de forma lúdica e envolvente. 

O projeto combina mecânicas de aventura com desafios de vocabulário e gramática, focado em estudantes que estão iniciando sua jornada no idioma.

## 📜 A História

No mundo mágico de **Sharimar**, viviam três grandes amigos: **Nikki, Noah e Gilliard**. No entanto, um desentendimento rompeu essa amizade. Gilliard, movido pela raiva, raptou Noah — o namorado de Nikki.

Agora, Nikki precisa enfrentar inúmeros desafios e resolver as charadas propostas por Gilliard para salvar seu amado. O único caminho para o resgate é o conhecimento: cada nível superado representa um novo aprendizado em inglês que aproxima o casal novamente.

## 🎯 Objetivo Educativo

O jogo foca em:
- **Vocabulário básico:** Objetos, cores e direções.
- **Gramática inicial:** Estruturas de frases simples e saudações.
- **Interpretação:** Resolver puzzles baseados em comandos em inglês.

## 🚀 Tecnologias Utilizadas

- **Motor de Jogo:** [Godot Engine 4.2](https://godotengine.org/)
- **Linguagem:** GDScript
- **Estilo Visual:** Pixel Art
